 Tic Tac Toe Game

This project is a Python-based Tic Tac Toe game where the user plays against the computer.

The computer uses Python's random module to generate its moves.

Developed as part of my internship at HexSoftwares.

---

Features

✔ Player vs Computer  
✔ Random AI move logic  
✔ Win detection  
✔ Draw detection  
✔ Clean terminal-based board display

 Technologies Used

- Random module
- Functions
- Lists
- Conditional logic

 How to Run

Open terminal and run:

py -3.10 tictactoe.py

---

 How to Play

Board positions are numbered as:

1 | 2 | 3  
--+---+--  
4 | 5 | 6  
--+---+--  
7 | 8 | 9  

Enter numbers (1–9) to place your move (X).  
Computer automatically places O.
